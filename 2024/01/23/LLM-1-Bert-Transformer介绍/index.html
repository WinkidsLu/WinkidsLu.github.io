LLM (1):Bert&Transformer介绍

发表于2024-01-23|更新于2024-02-11|LLM

1. BERT(Bidirectional Encoder Repersentation from Transformers)

Google AI Language: BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding

1.1 bert架构

  • bert架构: 双向transfromer-encoder架构，前后都有连接
  • GPT架构: 单向transformer-encoder架构，只向后面连接
  • ELMo架构: LSTM架构，left-to-right and right-to-left LSTM

NLP的两大任务：理解与生成
时间线：Neural Language Model(2001) -> Multitask Learning(2008) -> Word Embedding(2013) -> Neural Network for NLP(2013) -> Sequence2Sequence Model(2014) -> Attention Mechanism(2015) -> Pretrained Language(2018)
前transformer时代: RNN, LSTM, GPT
后transformer时代，统一的范式：Pre-training + Fine-tuning
数据收集预处理 -> model design -> 预训练（无标注的收集数据）-> 微调（任务相关的数据，一般需要标注，例如情感识别）

1.2 BERT的三个模块：

embedding + transformer-encoder + 预微调模块

1.2.1 embedding

分为三种embedding

- Token Embedding：字词级别的embedding
- Segment Embedding：标记这个词属于哪一句话，假设输入有多句话
- Position Embedding：标记词的位置。**与transformer不同，transformer使用三角函数标记词的绝对位置；而bert是通过学习得到的**

三种embedding通过相加得到最终的embedding representation

1.2.2 预微调模块

预微调模块，对应四种任务。对于sequence-level的分类任务，直接取[cls]对应的hidden state，后接全连接层进行分类。不同的任务是通过微调预微调模块来实现的
(1) sentence-pair classification: 预测两个句子的关联程度
(2) single sentence classification: sentiment analysis
(3) Q-A task: 例如阅读理解，输出起始到终止字符的内容作为回答内容
(4) Single sentence tagging task: NER

1.2.3 BERT的超参数选择

  • batch size: 16，32
  • learning rate: 5e-5，3e-5，2e-5
  • epoch：3，4

1.2.4 BERT的两大预训练任务

Mask LM与NSP(Next Sentence Prediction)

Mask LM：带Mask遮掩的语言训练任务

  1. 原始训练文本中，随机选择15%的token作为参与MASK任务的对象
  2. 在被选中的token中，数据生成器不是将它们全部变成[MASK]，而是按照以下三种情况处理：
    1. 80%概率下用[MASK]替换token： my dog is hairy -> my dog is [MASK]
    2. 10%概率下用一个随机单词替换token：my dog is hairy -> my dog is cat
    3. 10%概率保持token不变
  3. 模型训练过程并不知道需要预测哪些单词，在高度不确定情况下学习token的上下文语义。同时由于只有15%的token参与了mask处理，并不会破坏原本语言的表达能力

Next Sentence Prediction(NSP): 下一句话预测任务

  1. NLP中有一类问题需要模型理解两个句子之间的关系，从而需要在模型训练中引入对应任务（如QA问题，NLI-Natural Language Inference）。BERT采用的方式是输入句子对（A，B），模型预测B是不是A的真实下句话
    1. 所有参与训练的句子记作A，50%的B是原始文本中真实跟随A的下一句话（isNext，代表正样本），50%的B是原始文本中随机抽取的一句话（noNext，代表负样本）
    2. 该任务中模型可以在测试集达到97%-98%的正确率

2. Transformer

Google: Attention Is All You Need

  • bert只用了encoder
  • gpt只用了decoder
  • BART和T5用了encoder+decoder
  • 关注训练数据大小以及参数量大小（million百万，billion十亿，trillion兆）

2.1 Encoder模块

经典transformer中的Encoder模块包括6个层层堆叠Encoder Block，每个Encoder Block包括一个多头自注意力层和一个前馈全连接层（Multi-head self-attention -> feedforward）* 6
